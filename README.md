#SmashBros Minecraft PaperMC Plugin

#Overview
* This is the fun SmashBros Game but in Minecraft plugin.
* Double jump/skip jump to move around your maps 
* Perform a Smash knocking players back
* Grab and Throw players 

More features to this plugin are yet to come! 

#Compatibility
Minecraft v26.1.2 For PaperMc Server  

#Current Features 
Double Jump - space + space (Skip jump occurs when looking straight)
Smash - double jump + shift
Grab - right click player
Throw - left click to throw player
Sound Effects 

Feel free to add onto this current implementation and have fun with it!
